# Промт для Replit

Создай современное веб-приложение (не просто HTML-страницу), которое заменит существующую Google-таблицу для управления рекламными роликами в торговых комплексах.

Основные требования:

## Общие требования

* Современный минималистичный интерфейс.
* Адаптивный дизайн.
* Светлая тема.
* Максимально быстрое отображение данных.
* Все перерасчеты происходят автоматически без перезагрузки страницы.
* Все изменения должны происходить в режиме реального времени.
* Код должен быть хорошо структурирован и легко расширяем.

---

# Верхняя панель

В верхней части страницы разместить следующие элементы:

### 1. Загрузка Excel

Кнопка:

> Загрузить Excel

Поддерживаемые форматы:

* .xlsx
* .xls

После загрузки данные автоматически импортируются.

Рядом разместить кнопку

> Отменить последнюю загрузку

которая полностью восстанавливает данные до предыдущего состояния.

---

### 2. Фильтр по городу

Мультивыбор.

Значения:

* Все города
* Москва
* Санкт-Петербург
* Казань
* Уфа
* Краснодар
* Красноярск
* Новосибирск
* Екатеринбург
* Ростов-на-Дону
* Самара
* Саратов
* Нижний Новгород

Должна быть возможность выбрать сразу несколько городов.

---

### 3. Формат торгового комплекса

Выпадающий список:

* Все ТК
* ГМ
* СМ

---

### 4. Примечание

Выпадающий список:

* Все
* Платник
* Не платник

---

### 5. Дата

Поле выбора даты.

После изменения даты должны автоматически пересчитываться все отображаемые данные.

---

# Основная часть страницы

Ниже должна отображаться группа карточек.

Каждая карточка соответствует одному городу.

Название города располагается сверху по центру.

---

## Внутри карточки

Отображается таблица.

### Строки

Каждая строка соответствует одному торговому комплексу.

Формат отображения:

```
ТК №123
(адрес)
```

---

### Столбцы

Каждый столбец соответствует одной рекламной кампании.

В ячейке отображается:

* название рекламной кампании;
* длительность ролика;
* заказчик.

---

Последний столбец называется

> Общий хронометраж

Он показывает сумму длительности всех роликов, которые отображаются в данной строке (для конкретного ТК).

Значение автоматически пересчитывается при любых изменениях.

---

# Работа с рекламными кампаниями

При выборе рекламной кампании должна быть возможность:

* удалить кампанию;
* удалить сразу несколько кампаний;
* после удаления автоматически пересчитать весь хронометраж.

---

# Информация о рекламной кампании

При выборе кампании отображается отдельная информационная панель.

В ней выводятся:

* название рекламной кампании;
* заказчик;
* длительность ролика;
* статус (платник / не платник);
* дата начала показа;
* дата окончания показа.

---

# Панель статистики

Разместить отдельный блок со статистикой.

Он должен автоматически пересчитываться после любого изменения данных.

Необходимо отображать:

* общий хронометраж;
* только платник;
* только не платник;
* только ГМ;
* только СМ;
* ГМ + СМ.

Все показатели рассчитываются относительно выбранной даты.

Главная задача — быстро понять, сколько секунд рекламы приходится на конкретный торговый комплекс в выбранный день.

---

# Импорт Excel

После загрузки Excel данные автоматически импортируются.

Структура файла:

Первая строка содержит заголовки.

Со второй строки располагаются данные.

Столбцы:

A — название рекламной кампании

B — дата начала показа

C — дата окончания показа

D — заказчик

E — примечание (платник / не платник)

F — длительность ролика (секунды)

G и далее — номера торговых комплексов.

Каждая заполненная ячейка после столбца F содержит отдельный номер ТК, на котором должен показываться данный ролик.

Если один ролик размещается в нескольких ТК, то он должен автоматически появиться во всех соответствующих строках.

---

# Хранение данных

Если бесплатная версия Replit поддерживает хранение данных, использовать встроенную базу данных или встроенное постоянное хранилище.

После загрузки Excel данные должны сохраняться на сервере.

Любой пользователь, открывший сайт, должен видеть последнюю загруженную информацию.

Данные должны быть доступны даже если компьютер пользователя, загрузившего Excel, выключен.

### 15. Справочник торговых комплексов ⭐⭐⭐⭐⭐

Хранить отдельный справочник ТК с полями:

* номер ТК;
* адрес;
* город;
* формат (ГМ/СМ).

Тогда Excel будет содержать только номера ТК, а адреса и принадлежность к городам будут автоматически подставляться из справочника. Это уменьшит количество ошибок и упростит поддержку системы.
